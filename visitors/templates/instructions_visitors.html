BOOKS


  1. Staff user can see library full books list .
  2. Visitor user can only see library full books list with no manipulets .
  3. Only staff user can make manipulets on all books in library (add , edit or delete).

LOANS


  1. Staff user can see library full books list .
  2. Staff user can see library full loans list of all visitors .
  3. Only staff user allowed to make loan for visitor. In this case , return date calculate according book type :
    • book type = 3 : loan must return in 10 days .
    • book type = 2 : loan must return in 5 days .
    • book type = 1 : loan must return in 2 days .
  4. Only staff user allowed to approve returning loan for visitor .

ACCOUNTS


  1. Visitor and staff users can login and logout to the site throw the main page .
  2. Visitor user can see and manpiulate only his personal details .
  3. Staf user can see visitors + staff details list .
  4. only Staff user can add another staff user .
